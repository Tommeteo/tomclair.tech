# Jarvis - Votre Assistant Intelligent

Site officiel de Jarvis, hébergé sur tomclair.tech.

## 🌐 Site Web

Le site est disponible à l'adresse : **https://tomclair.tech**

## 📋 Description

Jarvis est un assistant intelligent conçu pour vous aider dans votre vie quotidienne. Il peut gérer vos tâches, répondre à vos questions, contrôler vos appareils connectés et bien plus encore.

## ⚙️ Fonctionnalités

- **Assistant Vocal** : Interaction naturelle par la voix
- **Automatisation** : Automatisez vos tâches répétitives
- **Multi-Plateforme** : Accédez depuis n'importe quel appareil
- **Sécurisé** : Chiffrement de niveau militaire
- **Temps Réel** : Traitement instantané
- **Intégration API** : Connectez vos outils existants

## 🔧 Prérequis Techniques

- **Python 3.12.9 OBLIGATOIRE** (aucune autre version ne fonctionnera)
- Site web : HTML5, CSS3, JavaScript (interface uniquement)
- Backend : 100% Python

## 🚀 Installation

1. Installer **UNIQUEMENT** Python 3.12.9 depuis [python.org](https://www.python.org/downloads/release/python-3129/)
2. Double-cliquer sur `install.bat` ou exécuter : `pip install -r requirements.txt`
3. Configurer les API dans le fichier `.env`
4. Lancer Jarvis : double-cliquer sur `DEMARRER_JARVIS.bat`
